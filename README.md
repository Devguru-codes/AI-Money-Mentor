> **How it works:** User sends a query тЖТ **DhanSarthi** analyzes intent тЖТ delegates to the right specialist agent (e.g., tax question тЖТ KarVid, retirement тЖТ Yojana) тЖТ returns the combined response. All delegation is handled via the **OpenClaw multi-agent swarm** protocol.

## ЁЯдЦ The 9 Agents

| # | Agent | Role | Key Features |
|---|-------|------|--------------|
| 1 | **KarVid** ЁЯз╛ | Tax Wizard | Old vs New regime comparison, 80C/80D deductions, capital gains, Indian Tax Laws DB |

| 6 | **Vidhi** тЪЦя╕П | Compliance Officer | SEBI regulations, legal disclaimers, financial law lookup |
| 7 | **DhanSarthi** ЁЯза | Coordinator | Routes queries to the right agent via OpenClaw multi-agent swarm |
| 8 | **Life Event Advisor** ЁЯОЙ | Life Planner | Marriage, child birth, education тАФ goal-based SIP planning |
| 9 | **Couple's Planner** ЁЯТС | Joint Finance | Combined budgets, expense splitting, joint debt payoff strategies |

## ЁЯЪА Quick Start

### Prerequisites

- **Node.js** 18+ and npm
- **Python** 3.11+ (3.12 recommended)

### 1. Clone

```bash
git clone https://github.com/Devguru-codes/AI-Money-Mentor.git
cd AI-Money-Mentor
```

### 2. Backend Setup

```bash
cd backend
python -m venv venv

# Linux/Mac
source venv/bin/activate

# Windows
.\venv\Scripts\activate

pip install -r requirements.txt
uvicorn api_server:app --host 0.0.0.0 --port 8000
```

### 3. Frontend Setup

```bash
cd frontend
npm install
npx prisma generate
npx prisma db push
npm run dev
```

### 4. Open in Browser

- **Frontend:** `http://<your-ip>:3000` (Use `http://localhost:3000` if running locally)
- **API Docs:** `http://<your-ip>:8000/docs` (Use `http://localhost:8000/docs` if running locally)

## тЪЩя╕П Environment Variables

Create `.env` in the project root:

# OpenClaw Agent Swarm (configured via openclaw CLI on the server)

## ЁЯЪв Deployment (EC2)

The project runs on an AWS EC2 instance:

```bash
# SSH into the instance
ssh -i your-key.pem ubuntu@your-ip

# Start backend
cd ~/ai-money-mentor-unified/backend
source venv/bin/activate
nohup uvicorn api_server:app --host 0.0.0.0 --port 8000 &

# Start frontend
cd ~/ai-money-mentor-unified/frontend
nohup npm run dev -- -H 0.0.0.0 &
```

> **Note:** Ensure ports **3000** and **8000** are open in your AWS Security Group for public access.

## 💡 Recording Tips

1. **Toggle Dark Mode ON** before recording — click the moon icon in the navbar. Much better visual impact.
2. **Pre-login** with a test account so the navbar shows a username and dashboard stats populate.
3. **Full screen at 1080p** — the UI is designed for wide screens.
4. **Keep mouse movements smooth** — avoid jittery cursor.
5. **Wait for AI responses** — the streaming/loading animations fill the gap.
6. **Check green status dot** on DhanSarthi before starting — ensures backend is online.
7. **Don't rush page transitions** — let each agent page load fully before speaking.
8. **If an agent is slow**, narrate what it's doing: "Notice it's processing in real-time..."
9. **Some forms have defaults** — KarVid (₹12L income), YojanaKarta (₹50K expenses), DhanRaksha (₹1L income), and Niveshak (1 holding) come pre-filled. You can modify them to the values listed above, or use defaults if they look good on screen.
10. **Drop the "Regime Comparison" tab** in KarVid if time is tight — the calculator results alone are impressive.

## 🔁 Fallback Plan (If Backend is Slow)

If the backend takes too long on any agent response:
- Form-based agents (KarVid, YojanaKarta, DhanRaksha) have **frontend fallback calculators** — they work even if the backend is offline
- For chat-heavy agents (Life Event, Couple's, Vidhi), **pre-load** responses before recording by filling the form once, then refresh and re-submit on camera
- For BazaarGuru, if the stock API is slow, say "Live data takes a moment from NSE" while waiting
- For DhanSarthi, if the chat is slow, narrate the routing concept while the bouncing dots play
